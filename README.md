# 準備

# 実装時メモ
***

## 使用ライブラリ
* Express.js
* ejs (Embedded JavaScript templates)
  - テンプレートエンジン
  - https://ejs.co/
  - https://www.npmjs.com/package/ejs
* CSV parser for Node.js and the web
  - CSVファイルを読み込んでパースする

## 手順
1. > npm init
1. > npm install express
1. > npm install ejs
1. > npm install csv-parse
